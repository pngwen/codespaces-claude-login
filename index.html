Claude Code — Codespaces Login Helper

Step 1 — Install Claude Code

Run this command in your Codespace terminal to install Claude Code:

curl -fsSL https://claude.ai/install.sh | sh

Once installed, run claude and copy the authorization URL it gives you. Paste it below.
